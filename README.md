# 💰 Gestor Financeiro

> Aplicativo mobile simples e prático para controle de finanças pessoais.

---

## 📱 Sobre o projeto

O **Gestor Financeiro** é um aplicativo desenvolvido com o objetivo de ajudar usuários a organizarem suas finanças de forma rápida, intuitiva e eficiente.

A proposta do app é ser **simples e prático**, permitindo registrar e acompanhar receitas e despesas do dia a dia sem complicações.

Este projeto foi desenvolvido com foco em aprendizado e aplicação de tecnologias modernas no desenvolvimento mobile.

---

## 🎯 Objetivos

* Facilitar o controle financeiro pessoal
* Proporcionar uma experiência simples e intuitiva
* Aplicar conceitos de desenvolvimento mobile com React Native
* Integrar serviços em nuvem com Firebase

---

## 🚀 Funcionalidades

### ✅ Implementadas

## 🔐 Configuração do Firebase

Para executar o projeto corretamente, configure o Firebase:

1. Crie um projeto no Firebase
2. Ative o **Authentication**
3. (Opcional) Ative o **Firestore Database**
4. Crie um arquivo `.env` na raiz do projeto

Exemplo de configuração:

```id="env2"
FIREBASE_API_KEY=
FIREBASE_AUTH_DOMAIN=
FIREBASE_PROJECT_ID=
```
